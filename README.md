# To Do App 

This is a to do app with clean code and gulp.

# How To Use

Use the terminal to write this code.
### Clone this repository
```bash
git clone https://github.com/LamaSaad24/to-do-list-task3.git
```
### Install dependencies
```bash
npm install
```

### Run the app
```bash
npm start
```
